## Installation

Its [available in Hex](https://hex.pm/packages/tabebuia), the package can be installed
by adding `tabebuia` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:tabebuia, "~> 0.1.0"}
  ]
end
```

Documentation can be found at <https://hexdocs.pm/tabebuia>.
